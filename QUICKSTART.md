# Быстрый старт

## Установка (один раз)

```bash
# Установите зависимости
pip install -r requirements.txt

# Установите браузеры
playwright install chromium
```

## Использование

### Вариант 1: Интерактивный режим (самый простой)

```bash
python main.py
```

Скрипт запросит:
1. Номер телефона
2. Код из SMS (придет на телефон)
3. Поисковый запрос

### Вариант 2: С параметрами для React/Next.js разработчика

```bash
python main.py \
  --phone "+79991234567" \
  --search-role react_nextjs \
  --limit 20
```

Код из SMS будет запрошен интерактивно.

### Вариант 3: С кастомным сопроводительным письмом

1. Создайте файл `my_cover_letter.txt`:

```
Здравствуйте!

Меня заинтересовала вакансия. Имею опыт работы с React и Next.js.

Готов обсудить детали.

С уважением
```

2. Запустите:

```bash
python main.py \
  --phone "+79991234567" \
  --search-role react_nextjs \
  --cover-letter my_cover_letter.txt \
  --limit 15
```

## Примеры поисковых запросов

### React/Next.js разработчик

```bash
python main.py --phone "+79991234567" --search-role react_nextjs
```

### QA Lead

```bash
python main.py --phone "+79991234567" --search-role qa_lead
```

### Backend разработчик

```bash
python main.py --phone "+79991234567" --search-role backend
```

### Кастомный запрос

```bash
python main.py \
  --phone "+79991234567" \
  --search '(React OR Next.js) AND TypeScript NOT (стажер OR intern)'
```

## Что происходит при запуске

1. ✅ Открывается браузер
2. ✅ Выполняется вход через телефон и SMS
3. ✅ Выполняется поиск вакансий
4. ✅ Загружаются все результаты (автоматический скролл)
5. ✅ Собираются вакансии с кнопкой "Откликнуться"
6. ✅ Отправляются отклики (с сопроводительным письмом, если требуется)
7. ✅ Вакансии с тестами/вопросами пропускаются и скрываются

## Важные замечания

- ⚠️ Используйте разумные лимиты (не более 20-30 откликов за раз)
- ⚠️ Код из SMS обычно действителен ограниченное время
- ⚠️ Скрипт не может обойти капчу (если появится)
- ✅ Все данные обрабатываются локально
